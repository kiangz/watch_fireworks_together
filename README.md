# 一起看烟花 (Watch Fireworks Together)

一个实时烟花观看与弹幕交流的Web应用，使用FastAPI和WebSocket实现。

## 功能特性

- 🎆 实时烟花动画效果
- 💬 实时弹幕交流系统
- 🔄 自动同步历史弹幕
- 📱 支持多客户端同时连接

## 技术栈

- **后端**: FastAPI + WebSocket
- **前端**: HTML5 + JavaScript + CSS3
- **部署**: Uvicorn

## 项目结构

```
.
├── main.py          # FastAPI后端应用
├── index.html       # 前端页面
├── pyproject.toml   # 项目配置
```
